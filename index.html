DESCRIPTION
YouTube Live Stats is a simple web application that makes use of tools such as the YouTube Data API and Chart.js to do a live tracking of YouTube channels, in a visually pleasing manner.
FEATURES
You can start adding channels as soon as possible using our custom built search feature. Search the channel by it's channel URL, channel username, or channel ID. Once you add a channel, a new tracking element will appear, allowing you to view simple information on the channel, along with a live subscriber count tracker. You can add as many channels create as many tracking elements as you want!
DISCLAIMER: As of August 2019, the YouTube API has changed the way they reported subscriber counts, rounding to the nearest thousands for all channels with over 1000 subscribers.
